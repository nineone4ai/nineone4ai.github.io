# PRD 分析 CheckList

> 用于检查 PRD 分析完整性的清单，确保不遗漏关键字段和逻辑

---

## 用途

- **评审会前**：快速过一遍，标记不确定项
- **评审会后**：逐项确认，确保分析完整性
- **开发过程中**：作为自测清单

---

## 一、字段完整性检查

### 列表页字段

- [ ] 显示字段（与原型逐一核对）
- [ ] 筛选条件字段（精确筛选、模糊搜索、范围筛选）
- [ ] 排序字段（默认排序、可切换排序）
- [ ] 分页配置（每页条数、页码跳转）
- [ ] 列表操作项（查看、编辑、删除、更多）
- [ ] 批量操作（批量删除、批量导出、批量审批）

### 表单页字段

- [ ] 必填字段标识
- [ ] 选填字段标识
- [ ] 字段类型（文本、数字、金额、日期、下拉、单选、多选、文件上传）
- [ ] 字段长度限制
- [ ] 字段默认值
- [ ] 字段只读/可编辑状态
- [ ] 字段联动关系（A 变 B 变）
- [ ] 字段校验规则（格式、范围、唯一性）

### 实体字段

- [ ] 主键 ID
- [ ] 基础业务字段
- [ ] 外键关联字段
- [ ] 冗余字段（用于查询/展示）
- [ ] 审计字段（创建人、创建时间、更新人、更新时间）
- [ ] 软删除标记

---

## 二、按钮完整性检查

### 列表页按钮

- [ ] 新建按钮
- [ ] 编辑按钮
- [ ] 删除按钮
- [ ] 查看/详情按钮
- [ ] 导出按钮
- [ ] 导入按钮
- [ ] 批量操作按钮

### 详情页按钮

- [ ] 保存按钮
- [ ] 取消按钮
- [ ] 提交按钮
- [ ] 审核按钮
- [ ] 更多操作

### 按钮定义要素

每个按钮需要明确：
- [ ] 按钮类型（主按钮、次按钮、危险按钮、链接）
- [ ] 权限标识（谁能看到）
- [ ] 操作前置条件（什么条件下可点）
- [ ] 点击后行为（跳转、弹窗、调用接口）
- [ ] 二次确认（是否需要确认）

---

## 三、业务逻辑检查

### 业务规则

- [ ] 金额计算规则（精度、四舍五入）
- [ ] 状态流转规则（状态机）
- [ ] 数据权限规则（可见范围）
- [ ] 操作权限规则（谁能做什么）
- [ ] 业务约束条件（前置条件、后置条件）

### 数据联动

- [ ] 字段间联动（A 变化触发 B 更新）
- [ ] 跨表联动（本表变化触发他表更新）
- [ ] 自动计算字段（公式推导）
- [ ] 数据同步（多端数据一致性）

### 状态说明

- [ ] 所有状态枚举值
- [ ] 状态含义说明
- [ ] 状态变更触发动作
- [ ] 状态流转图

---

## 四、流程图检查

- [ ] 业务流程图（用户操作视角）
- [ ] 数据流程图（系统处理视角）
- [ ] 时序图（关键交互）
- [ ] 状态流转图

---

## 五、API 接口检查

### 接口清单

- [ ] 查询接口（列表、详情）
- [ ] 创建接口
- [ ] 更新接口
- [ ] 删除接口
- [ ] 导入/导出接口
- [ ] 关联接口（下拉数据源）

### 接口参数

- [ ] 请求参数完整性
- [ ] 响应字段完整性
- [ ] 错误码定义
- [ ] 分页参数
- [ ] 排序参数

---

## 六、数据模型检查

### ER 图

- [ ] 所有实体已识别
- [ ] 实体关系已定义（1 对 1、1 对多、多对多）
- [ ] 主键、外键已标注

### 表结构

- [ ] 字段类型、长度已定义
- [ ] 必填/可选已标注
- [ ] 默认值已定义
- [ ] 字段注释已填写

### DDL

- [ ] CREATE TABLE 语句
- [ ] 索引定义
- [ ] 注释完整

---

## 七、异常场景检查

### 系统异常

- [ ] 网络失败处理
- [ ] 服务超时处理
- [ ] 数据并发冲突
- [ ] 数据不存在/已删除
- [ ] 系统错误提示

### 业务异常

- [ ] 校验失败提示
- [ ] 权限不足提示
- [ ] 数据重复处理
- [ ] 业务流程中断处理
- [ ] 回滚/撤销机制

---

## 八、权限检查

### 角色定义

- [ ] 所有参与角色清单
- [ ] 角色职责说明

### 功能权限

- [ ] 菜单可见性
- [ ] 按钮可见性
- [ ] 操作权限
- [ ] 数据范围权限

---

## 九、非功能性需求检查

### 性能要求

- [ ] 响应时间要求
- [ ] 并发量要求
- [ ] 数据量级

### 安全要求

- [ ] 敏感数据脱敏
- [ ] 操作日志记录
- [ ] 数据加密
- [ ] 防重复提交

### 兼容性要求

- [ ] 浏览器兼容
- [ ] 分辨率兼容

## 十、依赖关系检查

- [ ] 内部依赖（依赖其他模块/服务）
- [ ] 外部依赖（第三方接口）
- [ ] 数据依赖（基础数据准备）
